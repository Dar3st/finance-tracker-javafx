# Finance Tracker (JavaFX Desktop App)

**Краткое описание:** Десктопное приложение для учёта личных финансов с графическим интерфейсом.

**Стек технологий:** Java 21, JavaFX, JDBC, H2 Database.

**Функционал:**
*   Добавление/удаление доходов и расходов
*   Просмотр истории операций
*   Визуализация статистики по категориям

**Как запустить:**
1.  Клонировать репозиторий: `git clone https://github.com/Dar3st/finance-tracker-javafx`
2.  Открыть проект в IntelliJ IDEA
3.  Запустить класс `Main.java`
